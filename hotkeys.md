```markdown
# Atajos de teclado en VSCode

---

## Tabla rápida

| Atajo            | Función                            |
| ---------------- | ---------------------------------- |
| Ctrl + Shift + L | Seleccionar todas las repeticiones |
| Ctrl + L         | Seleccionar línea                  |
| Shift + Alt + A  | Comentar/Descomentar               |

## Selección

### Ctrl + Shift + L

Seleccionar todas las repeticiones de la palabra actual

Útil para: Cambiar múltiples variables con el mismo nombre a la vez

### Ctrl + L

Seleccionar línea completa

Útil para: Copiar, cortar o eliminar líneas rápidamente

---

## Comentarios

### Shift + Alt + A

Comentar/Descomentar selección

Útil para: Bloquear o desbloquear múltiples líneas de código

---

## Ejemplos de uso

### Seleccionar todas las repeticiones

1. Haz clic en una palabra
2. Presiona Ctrl + Shift + L
3. Todas las instancias se seleccionan
4. Escribe para reemplazar todas a la vez

### Seleccionar línea

1. Coloca el cursor en cualquier parte de la línea
2. Presiona Ctrl + L (puedes repetir para seleccionar más líneas)
3. Copia, corta o elimina

### Comentar bloque

1. Selecciona el código que quieres comentar
2. Presiona Shift + Alt + A
3. Se agrega // al inicio de cada línea
4. Presiona de nuevo para descomentar
```
